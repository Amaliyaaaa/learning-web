# learning-web-nodejs

В ходе изучения веб-разработки я выполняла различные тренировочные задания, которые для удобства собраны в двух репозиториях.
## Первая часть: основы
Первая часть [(learning-web-static: lab1-lab6)](https://github.com/Amaliyaaaa/learning-web-static) посвящена (по порядку и по нарастанию сложности) освоению основ HTML и CSS, основ верстки, расширенной настройки конфигурации nginx, применению grid и flexbox, основ JavaScript и работы с DOM, обработке событий, разработке таймеров, форм, базовой валидации на стороне клиента, а также реализовано получение данных по API (сторонние ресурсы) и хранение данных в браузере (cookies, localstorage).

## Вторая часть: работа с сервером и усложненная логика
Вторая часть (learning-web-nodejs: lab7-lab11) покрывает практическую реализацию следующих тем: основы node.js+fastify, файловые операции node.js, базовая валидация в fastify, основы работы с БД (postgres): параметризованные запросы, транзакции, динамическое
построение запросов, и, наконец, аутентификация с JWT, авторизация. 

Все вышеперечисленное реализовано с помощью трех языков: HTML, CSS и JavaScript. Запускала и тестировала я с помощью локального сервера Docker, создавала отдельные контейнеры под разные цели.

### Как протестировать:
Я рекомендую скачать проект целиком. Запуск отдельных html-страниц возможен без дополнительных средств, но это даст вам мало, особенно во второй части - ее суть в реализации клиент-серверной архитектуры. Лучшим решением будет скачать и запустить docker, а затем запустить проект из среды разработки из папок-корешков проекта:
```
docker compose -f docker-compose-local.yml up
```
и
```
docker compose -f docker-compose.local.yml up --build
```
для первой и второй части соотвественно (проверяйте названия файлов). Я оставила все используемые мной параметры в конфигурациях, при необходимости вы можете их изменить под свои цели.
